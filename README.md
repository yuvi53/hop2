# Intorducting Hop2 a rust implementation [autojump](https://github.com/wting/autojump).
Well this thing is a work in progress some features work others don't, if it sounds something you can work with, feel free to use!

# How to make it work
1. Clone the repo.
2. Run command `cargo run --release` on it.
3. Place this snippit in your terminal `path/to/your/cloned/repo/hop.zsh/file`(again, you should always inpect scripts before doing stuff like this!)
